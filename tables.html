Table's Structure

A table can contain 3 parts, and a caption

  1. header: thead
  2. body: tbody
  3. footer: tfoot
  4. caption: caption

Create a Table

  • Create the table with the table element
  • Define each line with the tr (table row) element
  • Define each cell of the line with the td (table data) element

Add a caption

  • Add a caption with the tag: caption
  • caption tag should be the first element after the tabe tag
  • use align attribute to define the position of the caption (top, bottom, left, right)

Add a Header

  • define the header inside thead
  • create a table row with tr
  • define header text and attributes (i.e. colspan) with th: table header cell

Add a footer

  • Open the footer with tfoot
  • Creta the footer row with tr
  • Define content of footer cell with tf: table footer cell
